# Patch 012 — まばたき中間フレーム（half / almost）ピクセル定義

## 背景

設計書 v11 §4 では `open → half(60ms) → almost(60ms) → closed(06/90ms) → almost → half → open` と定義されているが、half / almost のピクセル座標は未定義。本 patch で定義する。

## 設計方針

- 目ソケット（4×8 at y=3〜y=10）を上下から対称に縮小して閉じを表現
- 閉じ目の合わせ位置は frame06 の y=7 に合わせる
- 瞳はソケットに収まる範囲で縮小し、ソケットが狭くなりすぎたら非表示

## ピクセル定義

### half（半閉じ）

目ソケットを上下各 2dot 縮小。瞳を 2×2 に縮小して中央配置。

| パーツ | サイズ | 位置（左/右） |
|--------|--------|---------------|
| 目ソケット（白） | 4×4 | 左(5, 5) 右(13, 5) |
| 瞳 | 2×2 | 左(6, 6) 右(14, 6) |

```
左目ソケット:           右目ソケット:
y5:  WWWW               y5:  WWWW
y6:  WPPW               y6:  WPPW
y7:  WPPW               y7:  WPPW
y8:  WWWW               y8:  WWWW
```

### almost（ほぼ閉じ）

目ソケットを上下各 3dot 縮小。瞳は非表示（ソケットが狭すぎるため）。

| パーツ | サイズ | 位置（左/右） |
|--------|--------|---------------|
| 目ソケット（白） | 4×2 | 左(5, 6) 右(13, 6) |
| 瞳 | なし | — |

```
左目ソケット:           右目ソケット:
y6:  WWWW               y6:  WWWW
y7:  WWWW               y7:  WWWW
```

### closed（frame06、既存）

| パーツ | サイズ | 位置（左/右） |
|--------|--------|---------------|
| 閉じ目線 | 4×1 | 左(5, 7) 右(13, 7) |

## タイミング

§4 定義に準拠:

| ステップ | 描画 | 保持時間 |
|----------|------|----------|
| 1 | half | 60ms |
| 2 | almost | 60ms |
| 3 | closed | 90ms |
| 4 | almost | 60ms |
| 5 | half | 60ms |

合計: 330ms（現行の簡易実装は 150ms）
